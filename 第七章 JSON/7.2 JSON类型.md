## 7.2 JSON类型
JSON名称/键值
```json
{"name" : "tunan"}
```
#### 
**JSON数字**
**
```json
{"age" : 30}
```
#### 
**JSON对象**
**
```json
{"name:"tunan","url":"www.tunan.com"}
```
#### 
**JSON数组**
**
```json
{
	"sites":[
    { "name":"小鸟" , "url":"www.xiaoniao.com" }, 
    { "name":"google" , "url":"www.google.com" }, 
    { "name":"微博" , "url":"www.weibo.com" }
  ]
}
```
#### 
**JSON布尔值**

```json
{"flag":true}
```
#### 
**JSON null**

```json
{"nan":null}
```
